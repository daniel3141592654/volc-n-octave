# volcán-octave
Hice un volcán en GNU Octave para un trabajo en clase. Usa el método numérico de Euler y lanza proyectiles aleatorios.

Así de interesante se pone.

El volcán es generado con la función `volcan.m`, que se genera con un cono, una función aleatoria y una interpolación
cúbica, posteriormente se calculan las trayectorias de los proyectiles con `proyectil.m`, y `getCircle.m` me sirve
para graficar un círculo por si se me olvida.
